## 🚀 How to Use

1. Clone the repository.

```bash
git clone https://github.com/NemisRuparel/Python.git
```

2. Open the notebook.

```bash
cd Python
jupyter notebook
```

or open **Python.ipynb** in **Jupyter Notebook**, **JupyterLab**, or **Visual Studio Code**.

3. Follow the notebook from top to bottom and run each cell to understand the concepts.
